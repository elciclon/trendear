# 📊 Financial Screener for Global Equities

A modular, data-driven platform designed to collect, store, analyze, and visualize key financial metrics for publicly traded companies worldwide. This project is being developed as an open-source tool, integrating data science, finance, and software engineering.

> 🎓 I am documenting the development process step-by-step on LinkedIn as part of a _public building_ and active learning exercise.

---

## 🚀 Project Goals

- 🏗 **Data Ingestion:** Build a backend to ingest and normalize data from quarterly and annual financial reports (Balance Sheet, Income Statement, Cash Flow).
- 🌍 **Global Coverage:** Support assets across different markets (U.S. Equities, CEDEARs, Merval, etc.).
- 🧮 **Financial Intelligence:** Automatically calculate and store key performance indicators (KPIs) such as P/E, ROE, FCF Yield, and more.
- 📊 **Exploration:** Enable interactive data discovery via a clean, web-based UI.
- 🧠 **Continuous Learning:** Serve as a practical, professional-grade project during my Data Science degree studies.

## 🧱 Architecture

- **Backend:** Python 3.11, PostgreSQL 16 + TimescaleDB.
- **ETL:** Pandas, SQLAlchemy, and custom scrapers tailored to various data sources.
- **KPIs:** Metrics calculated directly from raw financial data using SQL and Python.
- **Front-end:** In development (Angular or Dash).
- **Visualization:** Interactive charts and dynamic data tables.
- **License:** MIT.
